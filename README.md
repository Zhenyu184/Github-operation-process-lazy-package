## Linux安裝(Linux Installation)

### 「Debian系列安裝」

或其延伸發行版，如Deepin、Knoppix、Ubuntu

```
$ apt-get install git
```
### 「Red Hat系列安裝」

或其延伸發行版，如aLinux、CentOS、Fedora、openSUSE

```
$ yum install git-core
```
其他以打包方式套件管理的，如ArchLinux、Gentoo自己想辦法

### 檢查版本

```
$ git --version
```
